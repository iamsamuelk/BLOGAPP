
When the folder is dowmloaded to your machine, run through the following steps to view it

Install requirements
- In terminal type "pip install -r requirements.txt"

Activate environment
- In terminal type "source env/bin/activate"

Launch uvicorn local server
- In terminal type "uvicorn blog.main:app --reload"

Loading Swagger UI to view Project application
- Copy and paste "http://localhost:8000/docs" in your browser preferably Google Chrome
